# GenAIMagic Extension - Installation Guide

## Quick Install (Chrome)

1. **Download Extension**
   - Download the `genaimagic-extension.zip` file
   - Extract the ZIP file to a folder on your computer

2. **Open Chrome Extensions**
   - Open Google Chrome
   - Type `chrome://extensions/` in the address bar
   - Press Enter

3. **Enable Developer Mode**
   - Toggle the "Developer mode" switch in the top right corner

4. **Load Extension**
   - Click the "Load unpacked" button
   - Navigate to the extracted extension folder
   - Click "Select Folder"

5. **Done!**
   - You should see the GenAIMagic icon in your toolbar
   - If not, click the extensions puzzle icon and pin GenAIMagic

## Quick Install (Microsoft Edge)

1. **Download Extension**
   - Download the `genaimagic-extension.zip` file
   - Extract the ZIP file to a folder on your computer

2. **Open Edge Extensions**
   - Open Microsoft Edge
   - Type `edge://extensions/` in the address bar
   - Press Enter

3. **Enable Developer Mode**
   - Toggle the "Developer mode" switch in the left sidebar

4. **Load Extension**
   - Click the "Load unpacked" button
   - Navigate to the extracted extension folder
   - Click "Select Folder"

5. **Done!**
   - You should see the GenAIMagic icon in your toolbar
   - If not, click the extensions icon and pin GenAIMagic

## Using the Extension

### First Time Setup
No setup required! The extension connects to GenAIMagic.io automatically.

### Basic Usage
1. Browse to any webpage with content you want to rewrite
2. Click the GenAIMagic extension icon
3. Configure your preferences (word count, tone, audience)
4. Click "Analyze & Generate Content"
5. View and download your AI-generated content

### Best Results
- Use on pages with clear product information
- Works best on e-commerce sites, reviews, and articles
- Ensure the page has loaded completely before using

## Troubleshooting

### Extension Not Appearing
- Make sure Developer mode is enabled
- Try reloading the extension page
- Check that you selected the correct folder

### "Failed to Extract Content" Error
- Refresh the webpage and try again
- Some pages may block content extraction
- Try a different page or use the web app instead

### Content Generation Failed
- Check your internet connection
- The page might not have analyzable content
- Try using different configuration settings

## Uninstalling

### Chrome
1. Go to `chrome://extensions/`
2. Find GenAIMagic extension
3. Click "Remove"
4. Confirm removal

### Edge
1. Go to `edge://extensions/`
2. Find GenAIMagic extension
3. Click "Remove"
4. Confirm removal
